# API-Testing-Urban-Grocers-App-Simulacion-de-Caso-Real-

## 📌 Contexto del proyecto

Este proyecto consistió en validar la API de Urban Grocers, enfocada en la gestión de kits y servicios de entrega. El objetivo fue asegurar que los endpoints funcionaran correctamente y cumplieran con las reglas de negocio.

---

## 🔍 Mi análisis

Analicé la documentación de la API usando ApiDoc e identifiqué los parámetros críticos:

- deliveryTime
- productsWeight
- productsCount

Diseñé y ejecuté 45 casos de prueba utilizando Postman aplicando:

- Partición de equivalencia
- Valores límite
- Escenarios positivos y negativos

Validé:

- Códigos HTTP (200, 400)
- Respuestas JSON
- Reglas de negocio

---

## ✅ Conclusiones

Se detectaron defectos funcionales y se reportaron en Jira.  
Las pruebas ayudaron a mejorar la estabilidad y calidad de la API.

El sistema ahora cumple correctamente con los requisitos funcionales.
